# 🍕 Pizza Ordering Application

The **Pizza Ordering Application** is a RESTful backend application developed using **Java** and **Spring Boot**. It allows users to browse pizzas, place orders, manage customer information, and track order details through secure and efficient REST APIs. The application follows a layered architecture using Controller, Service, and Repository layers to ensure clean, maintainable, and scalable code.

## 🚀 Features

* Customer registration and management
* Pizza menu management (Add, Update, Delete, View)
* Place and manage pizza orders
* Order status tracking
* CRUD operations using REST APIs
* Request and Response DTO implementation
* Exception handling and input validation
* Database integration using Spring Data JPA
* API testing with Postman

## 🛠️ Technologies Used

* Java 17
* Spring Boot
* Spring Web
* Spring Data JPA
* Hibernate
* MySQL
* Maven
* Lombok
* Postman
* Git & GitHub
* GitHub Copilot

This project demonstrates backend development skills, REST API design, database integration, layered architecture, and clean coding practices using modern Java and Spring Boot technologies.
